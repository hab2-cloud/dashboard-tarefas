# Dashboard de Tarefas Inteligente

## Objetivo
Sistema de gerenciamento de tarefas com operações CRUD.

## Tecnologias
- HTML
- CSS
- JavaScript

## Funcionalidades
- Adicionar tarefas
- Listar tarefas
- Editar tarefas
- Remover tarefas

## Estrutura
- index.html → estrutura
- style.css → design
- script.js → lógica

## Armazenamento
Os dados serão armazenados no localStorage.
